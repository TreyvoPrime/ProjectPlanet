# Project Planet

Project Planet is a standalone browser game about opening cosmic packs, collecting celestial cards, and battling random opponents.

## What It Includes

- 4-card pack openings with weighted rarity odds
- Collection gallery with rarity glow styling and generated card art
- Local save data with wins, losses, daily free pack, duplicates, and aura leveling
- Animated battle overlays with stat-by-stat comparisons
- No Discord bot, no OAuth, and no required secret environment variables

## Run Locally

```powershell
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```powershell
npm run build
npm run start
```
